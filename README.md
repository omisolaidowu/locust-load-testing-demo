# Locust Load Testing Demo

This repository is a simple demo of how to perform load testing with Python's Locust library.

## Project structure

```
api_example
├──locustfile.py
├──main.py
├──requirements.txt
└──.gitignore
```

### To run the test:

#### Install the requirements:

```
pip3 install -r requirements.txt
```

#### Run the locust file:

```
locust
```
